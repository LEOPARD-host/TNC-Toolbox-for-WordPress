== Frequently Asked Questions ==

= Does the plugin allow me to purge the NGINX User Cache for an account? =

Yes, it does! This can easily be done via the button in the admin top bar.


== Installation ==

1. Go to `Plugins` in the Admin menu
2. Click on the button `Add new`
3. Search for `WP Toolbox by TNC` and click 'Install Now' or click on the `upload` link to upload the ZIP
4. Click on `Activate plugin`
